# 📈 StockWiz: AI-Powered Stock Analysis Platform

**StockWiz** is a full-stack financial analysis tool designed to bridge the gap between raw market data and actionable insights. By combining real-time market data with Large Language Models (LLM), it provides users with automated, intelligent investment summaries.

### 🚀 Key Features
* **Real-time Data Fetching:** Integrated **Yahoo Finance API** to provide live stock price updates and historical performance data.
* **AI-Driven Insights:** Leveraged **Gemini AI** to analyze market trends and generate automated "Buy/Sell/Hold" recommendations based on technical indicators.
* **Secure User Management:** Implemented robust authentication using **JWT (JSON Web Tokens)** and **bcrypt** for password hashing.
* **Scalable Backend:** Built with a modular Express.js architecture and MongoDB for flexible data storage.

### 🛠️ Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **AI Integration:** Google Gemini API
* **Data Source:** Yahoo Finance API (via `yahoo-finance2`)
* **Security:** JWT (JSON Web Tokens), Bcrypt
